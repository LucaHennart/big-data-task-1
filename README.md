# Big Data Task 1 - Spam detector project

- **Source code for the first exercise of Big Data Engineering course**
- **[Istructions available here](http://users.jyu.fi/~miselico/teaching/2017-2018/TIES438/exercises/exercise1/)**
- **[Datatsets available here](http://www2.aueb.gr/users/ion/data/enron-spam/)**
## Emails

- Emails have to be placed into the emails/**[1-6]** folder.
- Dataset 6 contains only emails to be queried.
- QueryFiles should contain a subset of Dataset 6.
- dq/dq.txt contains a single email to be queried.

**For each dataset email folder, you should place spams and genuine emails together**
